CSS คืออะไร?

CSS ย่อมาจาก "Cascading Style Sheet"
คือภาษาที่ใช้เป็นส่วนของการจัดรูปแบบการแสดงผลเอกสาร HTML
โดยที่ CSS ใช้กำหนดรูปแบบของเว็บ ได้แก่ สีของข้อความ สีพื้นหลัง
ประเภทตัวอักษร และการจัดวางข้อความ
CSS สามารถกำหนดรูปแบบพร้อมกันทีเดียวได้ ทำให้เวลาแก้ไขไม่ต้องคอยแก้ทีละส่วน

การพัฒนา CSS

CSS เริ่มพัฒนาในปี ค.ศ. 1994
ถูกกำหนดมาตรฐานโดย W3C การใช้ CSS
ในการจัดรูปแบบการแสดงผล จะช่วยลดการใช้ภาษา HTML ในการตกแต่งเอกสารเว็บเพจ
ทำให้ซอสโค้ด (Source Code) ภายในเอกสาร HTML เหลือเพียงส่วนเนื้อหา ทำให้เข้าใจง่ายขึ้น

CSS1

1996 (พ.ศ. 2539)

CSS2

1998 (พ.ศ. 2541)

CSS3

2011 (พ.ศ. 2554)

ประโยชน์ของภาษา CSS


1.   ช่วยให้เนื้อหาภายในเอกสาร HTML มีความเข้าใจได้ง่ายขึ้นและในการแก้ไขเอกสารก็สามารถทำได้ง่ายกว่าเดิม
      เพราะการใช้ CSS จะช่วยลดการใช้ภาษา HTML ลงได้ในระดับหนึ่ง และแยกระหว่างเนื้อหากับรูปแบบในการแสดงผลได้อย่างชัดเจน

2.   ทำให้สามารถดาวน์โหลดไฟล์ได้เร็ว เนื่องจาก code ในเอกสาร HTML ลดลง จึงทำให้ไฟล์มีขนาดเล็กลง

3.   สามารถกำหนดรูปแบบการแสดผลจากคำสั่ง style sheet ชุดเดียวกัน
      ให้มีการแสดงผลในเอกสารแบบเดียวทั้งหน้าหรือในทุก ๆ หน้าได้ ช่วยลดเวลาในการปรับปรุง
      และทำให้การสร้างเอกสารบนเว็บมีความรวดเร็วยิ่งขึ้น นอกจากนี้ยังสามารถควบคุมการแสดงผล
      ให้คล้ายหรือเหมือนกันได้ในหลาย Web Browser

4.   ช่วยในการกำหนดการแสดงผลในรูปแบบที่มีความเหมาะกับสื่อต่างๆ ได้เป็นอย่างดี

5.   ทำให้เว็บไซต์มีความเป็นมาตรฐานมากขึ้นและมีความทันสมัย สามารถรองรับการใช้งานในอนาคตได้ดี
