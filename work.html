การใช้งาน CSS

1. การเขียน CSS ในเอกสาร HTML

 การเขียนในเอกสาร HTML จะต้องเขียนใน Tag style
การเขียนแบบนี้ไม่เป็นที่นิยมเพราะถ้าเราต้องการแก้ไข ก็ต้องแก้ไขในทุกหน้าซึ่งจะทำให้เสียเวลา
ตัวอย่างการเขียน CSS ในเอกสาร HTML
< head >
< style >
* {
   color : pink ;
}
< /style >
< /head >

อีกแบบหนึ่งของการเขียน CSS ในเอกสาร HTML คือการเขียนใน HTML tag เลยดังตัวอย่าง
< p style= "color: blue ; margin-left: 20px;" >
   ตัวอักษรนี้สีฟ้า และห่างจากขอบทางขวามือ 20px
< / p >

2. การเขียน CSS แยกเป็นอีก 1 ไฟล์

 การเขียนแบบนี้เป็นที่นิยมมากเพราะถ้าเราต้องการแก้หน้าตาเว็บเพจของเราก็แก้ไฟล์ CSS
แค่ไฟล์เดียวก็สามารถทำให้หน้าเว็บเพจของเราเปลี่ยนไปได้หมด
ซึ่งการใช้งานก็ให้เราเขียน Code ที่หน้า HTML ดังนี้ ตัวอย่างคำสั่งที่ใช้นำเข้าไฟล์ CSS จากภายนอก
< head >
< link rel= "stylesheet" > type= "text/css" href= "ชื่อไฟล์ CSS ของเรา" >
< /head >

CSS Selector

Selector คือ HTML elements หรือ elements ที่ CSS Rule ถูกนำไปใช้ ประกอบด้วยแบบต่างๆ ดังนี้
1. *
* {
   color : pink ;
}

การใช้ * หมายถึง element ทุก element ของหน้าเว็บ
และข้างใน { } ใส่คำสั่งว่า color : red; หมายถึงตัวอักษรในทุก element จะเป็นสีชมพูทั้งหมด
2. .nameclass
.pink {
   color : pink ;
}

การใช้ .(ชื่อ class) คือ element ที่มี class=”ชื่อ class” ของหน้าเว็บ ให้แสดงลักษณะรูปแบบตามที่กำหนดใน css
จากตัวอย่างนี้ class="pink" จึงใช้ .pink แสดงว่า element ที่มี class=”pink” ของหน้าเว็บ ให้แสดงตัวอักษรสีชมพู
3. #nameid
#pink {
   color : pink ;
}

การใช้ #(ชื่อ id) คือ element ที่มี id=”ชื่อ id" ของหน้าเว็บ ให้แสดงลักษณะรูปแบบตามที่กำหนดใน css
จากตัวอย่างนี้ id="pink" จึงใช้ #pink แสดงว่า element ที่มี id=”pink” ของหน้าเว็บ ให้แสดงตัวอักษรสีชมพู
4. element
h2 {
   color : pink ;
}

element ต่าง ๆ ใน HTML สามารถนำชื่อ element นั้น ๆ มาเขียนใน css ได้เลย
จากตัวอย่างนี้ element h2 ให้แสดงตัวอักษรสีชมพู
5. and หรือ ,
h2, p {
   color : pink ;
}

จากตัวอย่างนี้ element h2 และ p ของหน้าเว็บให้แสดงตัวอักษรสีชมพู
6. element ใน element
div p {
   color : pink ;
}

จากตัวอย่างนี้ element p ที่อยู่ใน div ของหน้าเว็บให้แสดงตัวอักษรสีชมพู

ใส่ช่องว่างภายใน
กำหนดพื้นที่ตัวเองจากขอบ จะมีพื้นที่ว่างจากขอบมาถึงตัวเอง เมื่อกำหนดค่าให้
padding-top : กำหนดด้านบน
padding-right : กำหนดด้านขวา
padding-bottom : กำหนดด้านล่าง
padding-left : กำหนดด้านซ้าย
padding:บน ขวา ล่าง ซ้าย;

 padding : 10px 15px 10px 20px ;
ใส่ช่องว่างภายนอก
การผลักตัวเองออกจากขอบ ระยะห่างระหว่างขอบ
margin-top : ขอบบน
margin-right : ขอบขวา
margin-bottom : ขอบล่าง
margin-left : ขอบซ้าย
margin:บน ขวา ล่าง ซ้าย;

  margin : 10px 15px 10px 20px ;
ใส่เส้นขอบ
border-width: ความหนาของเส้นขอบ
border-style: กำหนดรูปแบบของเส้นขอบ (required)
border-color: กำหนดสีของเส้นขอบ
border: width style color;

  border : 5px solid #000 ;

1. เงา แนวนอน เลขบวกด้านขวา เลขลบด้านซ้าย
2. เงา แนวตั้ง เลขบวกด้านล่าง เลขลบด้านบน
3. ค่าการเบลอ เริ่มจาก 0
4. ขอบเขตการขยายวของการเบลอ (a spread distance)
5. สีของเงา

  box-shadow : 0 0 8px rgba(0, 0, 0, 0.15); ;

  float : right ;
เคลียร์ float
  clear : both ;

  overflow : hidden ;
ใส่ค่าความโปร่งทึบ
  opacity : 0.5 ;
แสดงผลแบบเต็มช่อง

แสดงผลต่อกันในบรรทัด

ปิดการแสดงผล
  display : none ;

  position : relative ;

การทำภาพเคลื่อนไหวด้วย CSS

@keyframes
@keyframes นั้นมีกฎที่ว่า style ของ element ในปัจจุบันจะเปลี่ยนไปตามที่เขียนไว้ใน @keyframes
@keyframes move {
   0% {
left : 0px ;
}
  100% {
left : 120px ;
}
}
และใน css property ของ class ที่ต้องการจะทำ animation จะต้องระบุชื่อของ animation และใส่ระยะเวลาของ animation ด้วย
animation-name : move ;
animation-duration : 3 ;
